# Udemy-Project
# 📊 Udemy Courses Data Analysis

## 📌 Overview
This project focuses on analyzing Udemy courses data to understand course trends, subscribers distribution, and insights about different course categories.

The project uses data analysis techniques to clean, explore, and visualize the dataset to extract useful information about Udemy courses.

---

## 🎯 Project Objectives
- Explore the Udemy courses dataset.
- Analyze different course subjects and their popularity.
- Find relationships between courses and number of subscribers.
- Visualize data to make insights easier to understand.
- Practice data cleaning and exploratory data analysis (EDA).

---

## 📂 Dataset
The dataset contains information about Udemy courses including:

- Course Subject
- Course Level
- Number of Subscribers
- Course Details and other related features

Dataset file:
`udemy_courses_clean Dataset.csv`
